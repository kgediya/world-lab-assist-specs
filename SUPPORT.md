# Support

For help with this project:
- review the README for setup and wiring notes
- verify Lens Studio inspector references are assigned correctly
- verify the Supabase Edge Function and project URL are configured correctly
- test on Spectacles for device-only camera behavior

If you are troubleshooting generation:
- confirm the World Labs API key is set in the setup panel
- confirm the selected model is valid
- confirm the Edge Function accepts the `apiKey` and `modelName` fields from the lens

For security-sensitive issues, use the process described in `SECURITY.md` instead of posting sensitive details publicly.
