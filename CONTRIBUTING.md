# Contributing

Thanks for contributing to WLAO.

Project creator and maintainer: Krunal MB Gediya (Krazyy Krunal).

## Workflow

1. Work inside the `WLAO` project folder only.
2. Keep Lens Studio project files readable and version-control friendly.
3. Avoid committing local cache, user preference, or generated temporary files.
4. Test changes in Lens Studio preview when possible and on Spectacles for camera or device-only behavior.

## Coding Notes

- Lens Studio scripts in this project use JavaScript.
- Keep UI copy short, clear, and user-facing.
- Avoid logging raw secrets such as API keys.
- Prefer changing the active `WLAO` project rather than older copies.

## Pull Requests

- Describe the user-facing change.
- Mention any required scene wiring changes.
- Note whether the change was tested in preview, on Spectacles, or both.
